Instructions:

Run the batch file.
Enter the name of the text file you want to split (including the extension).
Enter the delimiter by which you want to split the file (e.g., newline, comma, tab).
The batch file will split the file into multiple parts based on the delimiter and save each part to a separate file with ".partX.txt" appended to the original filename, where X is the part number.
Press any key to exit.
